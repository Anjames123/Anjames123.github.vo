About the Project

This scientific calculator project was inspired by my passion for mathematics and the need for a comprehensive tool to handle complex calculations efficiently. The trigonometric functions, in particular, have always fascinated me, and I wanted to create a calculator that provides a seamless experience for exploring and utilizing these functions.

During my time at Holberton School, I recognized the opportunity to turn this passion into a portfolio project. The goal was not only to meet the requirements but to create a practical and visually appealing tool that showcases my skills in software development and design.

The development journey was challenging yet rewarding, pushing me to implement advanced mathematical algorithms and ensure a user-friendly interface. This project signifies my dedication to combining mathematical precision with elegant software design.
